# Installation Instructions

1. **Download the release [here.](https://github.com/sigmadev0202/radiocast4fivem/releases/tag/download)**
2. Unzip the folder and drag and drop **radiocast4fivem-download** into your servers **resources** folder.
3. Open **server.cfg** and put in
```txt
ensure radiocast4fivem
```
to make sure that radiocast starts when your server starts. 

## If you want the auto updater to restart the resource for updates, add 
```txt
add_ace resource.Radiocast4Fivem command allow
```
**to your server.cfg**
